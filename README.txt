Number Guessing Game
Overview
A Python game where the computer picks a random number between 1 and 100. Players guess the number, receiving clues after incorrect guesses. The score decreases with each wrong guess.

How to Play
Run the Script: Execute python number_guessing_game.py.
Make Guesses: Enter guesses when prompted. Follow the clues provided.
Score: Aim to guess the number with the fewest attempts.

Running in Docker
Build Image:
docker build -t python-numberguess .

Run Container:
docker run -it python-numberguess
